Typography & Design Tokens

Font Families

H1–H6, section titles, resume headers

Design Tokens

  • Root font size: 16px
  • Body font weight: 400
  • Heading font weight: 700
  • Line height: 1.7
  • Border radius (cards): 8px
  • Border radius (sections): 16px

  • Grid gap: 1.5rem
